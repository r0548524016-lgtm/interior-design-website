# 🏛️ Avital Interior Design - Responsive Website

A modern, elegant, and fully responsive multi-page website built for an interior design studio. The website showcases design portfolios across various categories including kitchens, living rooms, and office spaces.

## ✨ Features & Highlights

- **Hero Video Background:** Immersive video integration on the homepage landing section.
- **Dynamic Portfolios:** Multi-page navigation (`Kitchens`, `Living Rooms`, `Offices`) with specialized layouts.
- **Interactive Design Tools Section:** Embedded external tools for user inspiration and planning.
- **Contact Form & Map Integration:** Integrated contact section with interactive Google Maps iframe.
- **Fully Responsive:** Custom media queries ensuring optimal display across mobile, tablet, and desktop screens.

## 🛠️ Technologies Used

- **HTML5:** Semantic layout structuring (`<nav>`, `<section>`, `<footer>`, `<table>`, `<iframe>`, `<video>`).
- **CSS3:** Advanced styling using:
  - **CSS Grid:** Dynamic gallery grid layouts with auto-fill/auto-fit capabilities.
  - **Flexbox:** Seamless navbar alignment and flex-wrap impression cards.
  - **Transitions & Hover Effects:** Smooth scaling, backdrop-blur filters, and custom shadow effects.
  - **Media Queries:** Responsive break-points (`@media (max-width: 768px)`).
